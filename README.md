# CodeAlpha Hotel Reservation System

A command-line Core Java application engineered to handle booking operations, manage room lists, organize customer files, and generate business reports.

## Features List
1. **Room Management**: Register, view, update, and delete room listings.
2. **Customer Profiles**: Register, search, update, and delete guest accounts.
3. **Reservations Booking Engine**: Book rooms, cancel bookings, handle check-in/checkout, and run automatic billing calculations.
4. **Business Analytics Dashboard Reports**: Output live lists of available/occupied rooms and run performance statistics.
5. **Sorting Operations**: Sort rooms, customers, or reservations using default Comparable and custom Comparator rules.
6. **Persistence Layer**: Auto-serializes objects to storage files.

## Minimum System Requirements
* Eclipse IDE.
* Java Development Kit (JDK) 17 or higher.
